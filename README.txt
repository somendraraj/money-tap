Money-tap bit coin api

Prerequisite: To run this application, first build the application using maven make sure Java 8 and Maven installed to build it.

Goto project root directory and execute: $ mvn clean install

Run the application:

Goto project root directory and execute: $./startup.sh to start the application in interactive mode

wait for one minute or check the logs in console or hit the below api using postman/browser

in console logs you will get post data which is being generated in every 1 minute

API list : http://localhost:8085/bitcoin/list
